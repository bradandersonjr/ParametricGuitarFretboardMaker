KEY FEATURES
================================================================================

FIND ITEMS
  - By exact name (case-sensitive)
  - By regex pattern (e.g., "^Fret.*")
  - Get all items in a group
  - Check if item is a group

SUPPRESS/UNSUPPRESS
  - Single items
  - Multiple items by pattern
  - Group + all contents (cascading)
  - Only group contents (not the group itself)

QUERY STATE
  - Total items, active, suppressed counts
  - Group count and feature count
  - Detailed item state (type, index, rollup, size)
  - Timeline summary statistics

ERROR HANDLING
  - Try-catch on all operations
  - Logged to Fusion console
  - Returns success/failure codes
  - User-friendly error messages

PERFORMANCE
  - O(n) list operations (n = timeline items)
  - Typical design (50-200 items): milliseconds
  - Regex pattern support for efficient batch ops


USAGE PATTERNS
================================================================================

1. QUICK HIDE:
   suppress_item(design, "Fret Slot Cuts")

2. HIDE GROUP + CONTENTS:
   suppress_group_with_contents(design, "Fret Slot Cuts")

3. HIDE BY PATTERN:
   suppress_items_by_pattern(design, r".*[Ss]lot.*")

4. SHOW EVERYTHING:
   unsuppress_items_by_pattern(design, r".*")

5. TOGGLE VISIBILITY:
   toggle_item(design, "Fret Slot Cuts")

6. GET STATS:
   summary = get_timeline_summary(design)
   print(f"Active: {summary['active_count']}")

7. LIST ALL:
   items = get_all_items(design)
   for item in items:
       print(f"{item['name']} ({'Group' if item['type'] == 'Group' else 'Feature'})")
